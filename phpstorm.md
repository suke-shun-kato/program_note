# ショートカット

| |mac|windows|
|:---:|---:|---:|
|対応括弧へ移動|control + M||
|行ジャンプ|command + L|ctr + G|
|単語選択|option + ↑||
|メソッド選択|option + ↑ -> ↑||
|複数行選択（マウス、飛び飛び）|option + クリック||
|複数行選択（マウス、連続）|shift + option + ドラッグ||
|複数行選択（キーボード）|option -> option -> ↑ or ↓||
|左のタブへ移動|command + shift + [||
|右のタブへ移動|command + shift + ]||
|一つ前の操作位置へ戻る|command ＋ [||
|   |command ＋ option ＋ ←||
|一つ先の操作位置へ戻る|	command ＋ ]||
|   |command ＋ option ＋ →||
